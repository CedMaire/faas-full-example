# FaaS Full Example

## Install
```bash
# Clone this repository and `cd` into the new folder.

npm install

npm install -g ts-node

# Fill `???`s in `main.ts` with your own values.
```

## Execute
```bash
ts-node main.ts basic
ts-node main.ts psbt
ts-node main.ts psbtni
```

## Read Source Code

```bash
cat main.ts
```
